## Tech Stack

**Client:** React, Next.js, TypeScript, TailwindCSS, Framer Motion, GSAP.

**Hosting:** Vercel.

[View Live Demo](https://vishal-folio.vercel.app/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/victorcodess/folio-v1
```

Go to the project directory

```bash
  cd folio-v1
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
